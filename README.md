## API Reference

#### Get class name

```http
  GET /nameClass?searchClass=${searchClass}&searchDay=${searchDay}&lesson=${lesson}
```

#### Get class in classroom

```http
  GET /class?searchClass=${searchClass}&searchDay=${searchDay}
```
